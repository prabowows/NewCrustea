# **App Name**: AquaControl Dashboard

## Core Features:

- Real-time Data Display: Visualize sensor data (load power, voltage, current, frequency, DO, temperature, pH, salinity) in real-time.
- Historical Data Charting: Display historical data for Dissolved Oxygen (DO) and other parameters, with customizable time ranges.
- Alarm Monitoring: Display real-time and historical alarms with details (originator, type, severity).
- Parameter Analysis: Display an interface for monitoring entity data and parameters, like DO, using interactive tabular arrangement.
- Aerator Control: Enable remote control of aerator with timeout request feature.
- Location Display: Show the pond location on a map with zoom functionality.

## Style Guidelines:

- Primary color: Moderate cyan (#46B2E0) for a clean and technological feel.
- Background color: Light cyan (#E0F7FA), providing a clean and neutral backdrop for data visualization.
- Accent color: Muted blue (#5B86E5), to subtly highlight interactive elements and key data points without overwhelming the design.
- Body and headline font: 'Inter' (sans-serif) for a modern and easily readable interface.
- Use simple, outlined icons from Material Design to represent different data types and functionalities.
- Use a responsive grid layout to organize data cards and charts. Prioritize key metrics at the top and allow for easy customization of displayed parameters.
- Incorporate subtle animations (e.g., data loading, chart updates) to enhance the user experience without being distracting.
